# 🚨 SafeSpeak – Voice-Based Emergency Alert System

⭐ AI-powered emergency alert system using voice recognition and real-time webhook integration

---

## 📌 Overview

SafeSpeak is a voice-enabled emergency alert system that allows users to quickly send alerts using speech. It is designed for situations where typing or making a call is difficult, enabling faster and hands-free communication with emergency services or support systems.

---

## ⚠️ Problem Statement

In emergency situations, users may not have the ability to type messages or make calls. This delay can be critical. There is a need for a fast, voice-based alert system that can instantly detect danger and notify the appropriate system.

---

## 💡 Solution

SafeSpeak uses voice recognition and AI to:

* Capture user speech
* Detect emergency-related keywords
* Classify severity (High / Normal)
* Send real-time alerts to backend via webhook

---

## 🔄 Workflow

User → Voice Input → AI Agent → Backend → Alert Generated

---

## 🛠️ Tech Stack

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js, Express
* AI Integration: Bolna Voice Agent
* API: Webhook (Custom Function)

---

## 🚀 How to Run the Project

### 🔹 Backend Setup

```
cd backend
npm install
node server.js
```

### 🔹 Frontend Setup

```
Open frontend/index.html in your browser
```
